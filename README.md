# GugleAI

[English](README.en.md) | 简体中文

</div>

一个基于 Tauri 2 + Vue 3 的桌面图像生成客户端，适配 OpenAI 图像接口（如 `gpt-image-2`）及兼容的第三方中转服务。

## 功能

- **多工作区**:左侧窄导航栏可在图像生成、文字聊天和无尽画布之间切换,三个工作区由独立 Vue Router 页面承载
- **提供商与模型**:可在模型设置右侧直接添加、命名和编辑提供商,保存各自的 API 地址、API Key 和模型列表;未保存时切换页面会提示保存、放弃或继续编辑;模型支持 ID、显示名称、介绍、生图标记和上下文长度,通过独立弹出层添加或编辑
- **分类设置**:设置页面使用独立侧边栏和子路由,分为模型设置、通用设置和全屏日志;每次从主导航进入设置时默认打开模型设置,其中另有提供商侧边栏
- **通用与界面**:可在通用设置维护用于提示词变量的用户名称,并选择浅色、深色或跟随系统主题;首次运行会生成“用户+四位随机数”的默认名称;界面控件与颜色统一使用 Arco Design,跟随主题变量实时切换
- **智能体设置**:可在智能体设置中修改无尽画布“提示词生成”的系统提示词,维护聊天智能体并在聊天底部切换;默认助手可编辑,也可添加和删除自定义智能体
- **提示词变量**:智能体系统提示词支持 `{{date}}`、`{{time}}`、`{{datetime}}`、`{{system}}`、`{{arch}}`、`{{language}}`、`{{model_name}}` 和 `{{username}}`,请求发出前按当前环境及通用设置中的用户名称动态替换
- **文字聊天**:通过 `/chat/completions` 进行多会话文字对话;聊天记录持久化,支持新建、重命名、删除和复制消息,助手消息显示实际模型;首轮完成后可用指定模型异步生成标题,也可关闭标题生成
- **分组模型选择**:生图、聊天、对话标题和无尽画布节点均按提供商分组选择模型,选中后显示提供商标签,请求自动使用模型所属提供商的地址和 Key
- **无尽画布**:进入时从画布库选择或新建空白画布,节点、图片、连线和视口按画布持久化,编辑器可返回画布库;默认以 35% 缩放打开新画布;手动实线与自动虚线连线使用不同样式,双击可断开手动连线;文字生成会创建并自动连接新的文字子节点;生成时只读取直接相连的上一级节点,文字节点内容加入提示词,图像节点作为参考图;每个参考图节点只保存一张图片并隐藏无关的模型设置;节点图片支持右键复制提示词、复制图片和保存
- **粘贴导入配置**:直接 Ctrl+V 粘贴连接配置自动添加并切换到对应连接,支持 newapi 渠道 JSON(`{"_type":"newapi_channel_conn",...}`)和 Codex CLI `config.toml`(提取 `base_url`)
- **文生图**:输入提示词调用 `/images/generations` 生成图片
- **参考图编辑**:添加任意数量的参考图,支持文件选择、拖拽和剪贴板粘贴
- **接口模式**:自动 / Images 接口 / Chat 接口三种模式;自动模式下多参考图改走 `/chat/completions`,兼容 edits 只支持单图的中转服务
- **生图高级配置**:接口模式和请求重试集中在生图页面的高级配置中,默认折叠并由用户手动展开;错误码支持下拉多选并保存自定义选项
- **生成控制**:生成期间可停止当前任务,同时取消 API 请求、重试等待和结果下载
- **诊断日志**:记录生成请求的任务编号、阶段、耗时、请求体大小、代理状态和脱敏后的错误详情,便于排查超时及传输失败
- **生成参数**:模型、尺寸、生成数量和生成按钮集中在页面底部同一参数行,空间足够时贴底显示;支持自动、预设尺寸以及分别输入宽高的自定义尺寸
- **预览历史**:页面顶部使用独立滚动区显示紧凑缩略图,占用剩余空间并至少保留一行;空间不足时取消参数行贴底并允许整页滚动。每次生成的图片和提示词会追加并在重启后恢复,支持双击放大,以及右键复制图片或提示词、设为参考图、保存或删除图片,也可一键清空全部预览
- **结果保存**:兼容 `b64_json`、图片 URL(含相对路径)、Chat 内嵌 base64 等返回格式,一键保存到本地

## 开发

依赖:[Node.js 20.19+](https://nodejs.org/)、[pnpm](https://pnpm.io/)、[Rust](https://www.rust-lang.org/)。

前端使用 Vue Router Hash History 管理 `/image`、`/chat`、`/canvas` 和 `/settings` 页面,设置页进一步使用 `/settings/models`、`/settings/agents`、`/settings/general`、`/settings/logs` 子路由。页面组件位于 `src/views/`,设置子页面位于 `src/views/settings/`。模型编辑、未保存更改提示和预览交互弹层位于 `src/components/modals/`。Arco Design 组件通过 `unplugin-vue-components` 按需解析。

`src/App.vue` 只保留应用壳层、路由出口和全局弹层,`src/composables/controller/index.ts` 只负责组合依赖并向页面提供统一的视图模型。具体职责按模块拆分:

- `src/composables/`:提供商设置、应用设置、主题、日志、更新检查、结果历史,以及生图、聊天和画布工作区状态
- `src/services/`:OpenAI 兼容生成请求、重试与响应解析,以及 IndexedDB 预览历史和画布存储
- `src/domain/`:共享类型、默认数据和模型选择/规范化工具
- `src/api/`、`src/chat/`、`src/canvas/`:API 连接、聊天会话和画布图结构的领域对象
- `src/styles/`:按基础壳层、设置、生图、聊天、画布、结果弹层和响应式规则拆分的样式

所有网络请求统一经过 `src/composables/fetch/index.ts` 的 Tauri HTTP 封装,以应用系统代理;生成相关的 Images、Edits 和 Chat 请求由 `src/services/transport/index.ts` 统一处理。

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm tauri dev

# 打包
pnpm tauri build
```

## 使用

1. 在模型设置中添加提供商,直接在右侧填写名称、API 地址和 Key 并保存,再通过模型弹出层维护该提供商的模型列表
2. 生图、聊天和无尽画布从按提供商分组的下拉菜单选择模型;生图高级配置需手动展开
3. 聊天页面可新建和重命名多个持久化会话,并在底部选择智能体;智能体设置可编辑默认助手、添加自定义智能体和配置画布提示词生成规则
4. 从无尽画布的画布库选择或新建画布;文字节点生成后会新增自动连接的文字子节点,手动连线可双击断开;每个含图片的节点只保存一张参考图,可替换图片或连接多个参考节点到新的空图像节点进行生成
5. 图像预览会持续保留,双击图片可放大,右键可复制图片或提示词、设为参考图、保存或删除,也可清空全部预览
